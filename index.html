Algorithm Visualizer

Instructions
1. Choose from a selection of sorting algorithms to visualize from the dropdown menu.
2. Increase or decrease the amount of elements to sort using the size slider.
3. Increase or decrease the sorting speed of the algorithm using the speed slider.
4. Click sort and watch the algorithm come to life!
5. Click reset to receive a new set of randomized elements to sort.
